Simulasi Prinsip Kerja BLDC Motor — Medan Magnet & Rotor Stator

INPUT

V = Tegangan (Volt)
I = Kuat Arus (A)
N = Jumlah Lilitan
L = Panjang Kumparan (m)
f = Frekuensi (Hz)
P = Jumlah Pole

OUTPUT

Rumus yang digunakan:
B = μ₀×N×I / (2×L)
τ = N×F×r (Torsi)
ω = τ / b
RPM = ω×60 / 2π
P = τ × ω (Daya)
